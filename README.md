# Pizza Metrics Explorer

## Overview
Pizza Metrics Explorer is a comprehensive dashboard developed to monitor key sales and revenue metrics for a pizza business. This project utilizes **Python**, **Power BI**, and **MySQL Server** to provide real-time insights and automate reporting processes.

## Features
- **Real-Time Reporting**: Provides up-to-date sales and revenue data, helping businesses make informed decisions quickly.
- **Automated Workflows**: Reduces manual data compilation, saving 40 hours per month.
- **Upselling Opportunities**: Enhanced analysis led to a 20% increase in upselling opportunities.
- **Data Visualizations**: Includes intuitive charts and dashboards to track performance metrics.

## Technologies Used
- **Python**: For data processing and workflow automation.
- **Power BI**: For creating interactive and visual dashboards.
- **MySQL Server**: For efficient storage and retrieval of sales data.

## Impact
- **20% Increase in Upselling Opportunities**: By providing clearer insights into customer behavior, the dashboard helped identify key upselling opportunities.
- **40 Hours Saved Per Month**: Automation of data reporting significantly reduced manual labor and time spent on data compilation.
